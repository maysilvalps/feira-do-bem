# 🍅 Feira do Bem

A **Feira do Bem** é uma iniciativa digital com foco em sustentabilidade e combate ao desperdício de alimentos. O projeto conecta a identidade visual das feiras livres tradicionais a uma interface moderna, limpa e totalmente responsiva, incentivando ações práticas alinhadas ao consumo consciente.

## 🚀 Tecnologias Utilizadas

Este projeto foi construído utilizando tecnologias web fundamentais, priorizando leveza, acessibilidade e performance:

*   **HTML5:** Estruturação semântica das páginas principal e do diário/blog.
*   **CSS3:** Estilização personalizada baseada em variáveis de cor (conceito papel kraft/carimbo), design responsivo e adaptações avançadas para dispositivos móveis.
*   **JavaScript (ES6):** Manipulação de interações na interface, gerenciamento do menu responsivo e validações locais de formulários.

## 📦 Estrutura do Projeto

O repositório está organizado da seguinte forma:

*   `/css`: Arquivos de folha de estilo (`style.css`), contendo a identidade visual unificada.
*   `/images`: Banco de imagens do projeto, incluindo fotos de banner e o favicon personalizado (`icon`).
*   `/js`: Scripts de comportamento dinâmico e interações da interface.
*   `index.html`: Página principal da iniciativa.
*   `blog.html`: Seção de diário e novidades estruturada em formato clássico e interativo.

## 🛠️ Funcionalidades e Destaques

*   **Design Totalmente Responsivo:** Interface otimizada dinamicamente para computadores, tablets e smartphones.
*   **Identidade Visual Marcante:** Cores e elementos que remetem a feiras livres (tons terrosos, verdes e carimbos).
*   **Acessibilidade e Performance:** Código limpo, sem dependências pesadas ou scripts desnecessários, garantindo carregamento rápido.
*   **Favicon Customizado:** Ícone exclusivo integrado diretamente na aba do navegador.
